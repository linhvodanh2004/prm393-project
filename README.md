# Booking System

## Description
This project is a comprehensive booking system designed to facilitate room reservations and management for users, hosts, and administrators. It provides robust authentication, discovery, booking, payment, and administrative functionalities.

## Features

### I. User
- **Authentication & Security**:
    - **Registration/Login**: Supports session persistence and Refresh Tokens.
    - **Forgot Password**: Account recovery via Email.

- **Explore & Search**:
    - **Homepage**: View featured rooms, rooms near current location, and categorize by type.
    - **Advanced Search**: Quick search by name, location, price range, number of guests, and check-in/out dates.
    - **Smart Filters**: Filter by price (ascending/descending), rating, and amenities (Wifi, swimming pool, air conditioning, etc.).
    - **Map Integration**: Visual search via Google Maps integration.

- **Details & Booking**:
    - **Detailed Information**: View image carousels, descriptions, house rules, and reviews from past customers.
    - **Room Availability**: Check availability calendar and dynamic pricing based on dates.
    - **Booking**: Automatic total calculation, discount code application, and customer information confirmation.
    - **Booking Status**: Track process from Pending -> Confirmed -> Completed/Cancelled.

- **Payment & Management**:
    - **Multi-method Payment**: Supports VNPay, Credit Card, and COD (Cash on Delivery). Temporarily COD is the only allowed method.
    - **Booking Management**: View history, booking details, request date changes, or cancel bookings/refunds.
    - **Reviews**: Rate with stars, write comments, and upload real experience photos.
    - **Profile**: Update personal information, avatar, and change password.

### II. Host
- **Room Management**: Create, edit, delete rooms, and bulk update images/prices/statuses.
- **Booking Operations**: View guest list, confirm/reject requests, and chat directly with guests.
- **Calendar Management**: Proactively block/unblock dates on the calendar and set flexible pricing by time.
- **Revenue Reports**: View detailed monthly/annual statistics via charts and export reports.

### III. Admin
- **User Management**: Control User/Host lists, assign permissions, and lock violating accounts.
- **Room Moderation**: Review room content before publishing or hide non-standard rooms.
- **System Coordination**: Intervene to handle complaints, approve refunds, and control fraudulent transactions.
- **Notifications**: System sends confirmation notifications and automated emails for important events.
